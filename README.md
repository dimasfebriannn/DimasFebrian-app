# DimasFebrian-app (Laravel 12 Project)

Project ini merupakan tugas mata kuliah berbasis Laravel 12 yang berfokus pada pengembangan logika aplikasi dan UI tanpa menggunakan database.

## Prasyarat
Sebelum menjalankan project ini, pastikan perangkat Anda sudah terinstal:
* PHP >= 8.2
* Composer
* Node.js & NPM

## Langkah Instalasi

Ikuti langkah-langkah berikut untuk menjalankan project di lingkungan lokal:

   ```bash
   git clone [https://github.com/dimasfebriannn/DimasFebrian-app.git](https://github.com/dimasfebriannn/DimasFebrian-app.git)
   cd DimasFebrian-app/DimasFebrian-app
